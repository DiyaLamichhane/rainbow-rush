# 🌈 Rainbow Rush

A fast-paced color matching game where you catch falling colored blocks before they hit the ground!

## 🎮 How to Play

- Colored blocks fall from the top
- Click the matching color button at the bottom
- Match correctly = +10 points
- Miss 10 blocks = Game Over

## ⚡ Features

- Progressive difficulty (game gets faster as you score higher!)
- Combo system (chain matches for bonus points!)
- Power-ups (Lightning, Freeze, Magnet)
- Responsive design (works on phone, tablet, computer)
- Local leaderboard (saves your best score)

## 🎯 Controls

- **Desktop:** Click the color buttons with your mouse
- **Mobile:** Tap the color buttons with your finger

## 🏆 Scoring

| Action | Points |
|--------|--------|
| Normal match | 10 |
| 3x Combo | 20 |
| 5x Combo | 30 |
| 10x Combo | 50 |
| Power-up catch | 50 |
